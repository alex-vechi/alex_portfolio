## Projects

### [1. Fraud Detection - Risk Analyst Project](https://github.com/alex-vechi/risk_analyst_case)

**Project Overview:**  
Conducted analysis in Python and SQL to identify fraud patterns across thousands of payment transactions using an open financial dataset. Independently cleaned, processed, and analyzed transaction data, developing performance metrics to evaluate risk exposure. Designed preventive rules and documented findings in a comprehensive report and an interactive Colab notebook. Delivered actionable recommendations tailored for risk and product teams, with a focus on enhancing fraud prevention strategies and improving detection rates.

- **Technologies:** Python, SQL, Google Colab.
- **Highlights:**  
  - Explored transactional data and defined risk baselines
  - Identified behavioral and entity-based fraud patterns
  - Proposed prevention rules and strategies
  - Achieved a potential fraud reduction rate up to 75%
  - Full documentation in PDF and interactive notebook
